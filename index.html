Refining an AI-generated landing page into production quality

AI tools produce landing pages fast — and almost-right. This is the exact review pass I run to close the gap: responsiveness, semantics, accessibility, SEO, and performance. Every fix below is real code from this refactor.

Real scores from Chrome Lighthouse, mobile emulation. Both files — ai-draft.html and refactored.html — are in this repo; run Lighthouse yourself to verify.

1. Fixed-width layouts that break on mobile

The AI draft used pixel widths and absolute positioning. It looked perfect at 1440px and collapsed at 390px — text overflow, horizontal scrolling, overlapping cards.

✗ AI draft

<div style="width: 1200px; margin: 0 auto">
  <div style="position: absolute;
       left: 640px; width: 520px">
    <img src="hero.png" width="520">
  </div>
</div>

✓ Production

<div class="max-w-6xl mx-auto grid
     md:grid-cols-2 gap-12 items-center px-5">
  <img src="hero.webp"
       class="w-full h-auto" width="640"
       height="480" alt="Product dashboard
       showing weekly analytics" />
</div>

Why it matters: mobile-first grid + fluid widths means one layout system across all breakpoints. Explicit width/height attributes also eliminate layout shift (CLS 0.31 → 0.02).

2. Div soup instead of semantic HTML

Everything was a <div> — including the navigation, headings, and buttons. Screen readers see no structure; search engines see no hierarchy.

✗ AI draft

<div class="nav">
  <div class="nav-item" onclick="go()">
    Pricing</div>
</div>
<div class="big-text">Ship faster</div>
<div class="btn" onclick="signup()">
  Start free</div>

✓ Production

<nav aria-label="Main">
  <a href="#pricing">Pricing</a>
</nav>
<h1>Ship faster</h1>
<a href="/signup" class="btn">
  Start free</a>

Why it matters: real <nav>, <h1>–<h3> hierarchy, and links-as-links restore keyboard navigation, screen-reader landmarks, and crawlable structure. Semantic and contrast fixes took accessibility from 62 to 100.

3. Missing SEO & social metadata

The draft shipped with <title>Home</title> and nothing else. No description, no Open Graph, no canonical.

✓ Added

<title>Studioflow — Project Management for Creative Studios</title>
<meta name="description" content="Briefs, feedback and approvals in one place…">
<meta property="og:title" content="Studioflow — Project Management…">
<meta property="og:description" content="…">
<meta property="og:image" content="/og-cover.png">
<link rel="canonical" href="https://studioflow.example/">

Why it matters: pages built to run ads against live or die on link previews and search snippets. SEO score 82 → 100.

4. Performance: render-blocking everything

Five font weights, three unused JS libraries, and a 2.1 MB PNG hero. First Contentful Paint on throttled mobile: 4.8s.

  • ✓Hero PNG → WebP with explicit dimensions: 2.1 MB → 84 KB
  • ✓Dropped jQuery + two animation libraries; replaced with 12 lines of IntersectionObserver
  • ✓Fonts: 5 weights → 3, with display=swap and preconnect
  • ✓loading="lazy" on below-fold images and the map iframe

Result: Performance 88 → 96, with LCP as the remaining bottleneck to watch on image-heavy pages.

5. Forms that only pretended to work

The AI's email form had no labels, no validation, and no submit handling — a styled dead end.

✗ AI draft

<input placeholder="Email">
<div class="btn">Subscribe</div>

✓ Production

<form action="https://formspree.io/f/XXXX"
      method="POST">
  <label for="email" class="sr-only">
    Email address</label>
  <input id="email" name="email"
    type="email" required
    autocomplete="email">
  <button type="submit">Subscribe</button>
</form>

Why it matters: a real endpoint, native validation, an accessible label, and autocomplete. Conversion elements have to convert.

My AI-collaboration checklist

AI output is a strong first draft, not a deliverable. Before anything ships, I verify:

① Mobile-first at 360 / 768 / 1280px

② Semantic landmarks & heading order

③ Keyboard & screen-reader pass

④ Meta, OG, canonical present

⑤ Images sized, compressed, lazy

⑥ Forms wired, validated, labeled

⑦ Lighthouse ≥ 95 across the board

⑧ Copy placement matches the brief

This is exactly the "human-in-the-loop" role: AI for velocity, human judgment for quality.
